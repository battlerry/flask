入门演示

入门演示

示例代码：

# 导入类库
from flask import Flask
# 创建应用
app = Flask(__name__)
# 添加视图
@app.route('/')
def index():
    return 'Hello Flask!'
# 启动程序
if __name__ == '__main__':
    app.run()

启动参数

参数 说明
debug 是否开启调试模式，默认为False；开启后出错有调试信息，会自动加载文件。
threaded 是否开启多线程，默认是关闭的。
host 指定主机，默认为127.0.0.1，设置为'0.0.0.0'之后可以通过IP进行访问。
port 指定端口，默认为5000

启动示例：app.run(debug=True, threaded=True, host='0.0.0.0', port=5555)

访问测试

  • 在浏览器的地址栏中输入：http://localhost:5000/
