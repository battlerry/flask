Flask-Script

Flask-Script

说明

  • 简单来说，就是flask终端启动的参数解析器；这样就可以不改代码完成不同方式的启动。

安装

  • pip install flask-script

文档

  • 官方文档

使用

from flask_script import Manager

manager = Manager(app)

if __name__ == '__main__':
    manager.run()

参数

-?, --help                # 查看帮助
-h, --host                # 指定主机
-p, --port                # 指定端口
-d                        # 开启调试模式
-r                        # 自动加载

示例：python manage.py runserver -d -r -h 0.0.0.0 -p 5555
